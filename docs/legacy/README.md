# Legacy / roadmap docs

These documents describe the original Omnia vision: a knowledge ingestor that
collects context from Discord, GitHub, Jira and other sources into the memory
store. The packages exist in this repository (`internal/source/*`,
`internal/sink/*`, `internal/core`, `internal/enrich`) with passing tests, but
they are not yet wired into the `omnia` CLI.

They are kept here as the project roadmap — not as documentation of current
behavior. For the current CLI and cloud, see `DOCS.md` and `deploy/HOMELAB.md`
at the repository root.
